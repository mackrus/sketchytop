# Contributing

Thanks for considering contributing to `sketchytop`! 

## Development

1.  Clone the repository.
2.  Install the required dependencies (Rust/Cargo).
3.  Run `cargo build` to compile the project.
4.  Run `cargo test` to ensure all tests pass.
5.  Run `cargo fmt` to format the code.
6.  Run `cargo clippy` to check for common mistakes.

## Pull Requests

1.  Create a new branch for your changes.
2.  Make your changes and commit them with descriptive messages.
3.  Submit a pull request.
4.  Ensure that the CI checks pass.
